## LED Strip Demo
### Description
The LED strip consists of 24 LC8822-2020 adressable RGB LEDs that have dedicated Clock and Data pins and are compatible with Adafruit's DotStar library.
### Pins

LEDSTRIP_DATA
LEDSTRIP_CLOCK
